### 接口信息
- 策略名称：CategoryDdqGoodsList
- 接口名称：咚咚抢
- 接口版本：v1.2.1
- 接口地址：/category/ddq-goods-list

### 公共参数
|名称|类型|必须|说明|
| ------------ | ------------ | ------------ | ------------ |
|appKey|String|**是**|应用的唯一验证|
|version|String|**是**|API接口版本号|

### 请求参数
|名称|类型|必须|说明|
| ------------ | ------------ | ------------ | ------------ |
|roundTime|String|否|默认为当前场次，场次时间输入方式：yyyy-mm-dd hh:mm:ss|
