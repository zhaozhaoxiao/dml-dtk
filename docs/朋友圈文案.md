- 策略名称：FriendCircleList
- 接口名称：朋友圈文案
- 接口版本：v1.3.0
- 接口地址：/goods/friends-circle-list

| ------------ | ------------ | ------------ | ------------ |
|appKey|String|**是**|应用的唯一验证|
|version|String|**是**|API接口版本号|
